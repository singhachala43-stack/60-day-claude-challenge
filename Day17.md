# Day 17 – Vehicle Cost Analysis Dashboard

## Key Findings

* Petrol (E20): ₹5.65/km
* Monthly Fuel Cost: ~₹5,650
* E85: ₹6.30/km
* E85 Break-Even Price: ~₹76/L
* Petrol is currently the more cost-effective option

## Dashboard Features

* KPI Cards
* Fuel Cost Comparison
* Maintenance Analysis
* Environmental Impact Metrics
* SVG Charts & Gauges

## Learning

Fuel cost analysis should consider both fuel price and fuel efficiency. A cheaper fuel is not always the most economical choice.
