# Day 3 - Role-Based Prompting

## Challenge Objective

Learn how Role-Based Prompting changes AI responses by asking the same question from different personas and comparing the outputs.

## Question Asked

"How to engage new audiences in a mobile app?"

# Prompt 1: Without Any Role
How to engage new audiences in a mobile app?
Output Summary-
The response focused on general audience engagement strategies:

•Simplified onboarding
•Personalization
•Push notifications
•Referral programs

# Key Observation
The answer was broad, beginner-friendly, and applicable to most mobile apps.
# Prompt 2: Founder Persona
Imagine you are a founder. How would you engage new audiences in a mobile app?

# Output Summary
The response focused on business strategy and user understanding:

•Talk directly with users
• Find the product's "Aha Moment"
• Build a strong community
• Launch in stages
 
# Key Observation
The answer emphasized the "Why" behind engagement rather than specific implementation details.

# Prompt 3: Developer Persona
Imagine you are a developer. How would you engage new audiences in a mobile app?

# Output Summary
The response focused on technical implementation:

• Firebase Cloud Messaging (FCM)
• Deep Linking
•Analytics tracking
•Feature Flags
•A/B Testing

#Key Observation

The answer emphasized the "How" and included tools, technologies, and code examples.

# Comparison of Responses

| Aspect                 | General Response         | Founder Persona            | Developer Persona           |
| ---------------------- | ------------------------ | -------------------------- | --------------------------- |
| Focus                  | Engagement Tactics       | Business Strategy          | Technical Implementation    |
| Perspective            | User-facing              | Growth & Vision            | Engineering                 |
| Main Question Answered | What to do               | Why to do it               | How to build it             |
| Tools Mentioned        | Notifications, Referrals | NPS, Community, Retention  | Firebase, Amplitude, Sentry |
| Audience               | Beginners                | Founders & Product Leaders | Developers & Engineers      |
